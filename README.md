Payment Method Selector

A simple and interactive web application that allows users to select a subscription option and choose a preferred payment method. Built using HTML, CSS, and JavaScript, this project demonstrates form handling and DOM manipulation.

---

Features

* Subscribe using a checkbox
* Select a payment method:

  * MasterCard
  * Visa
  * PayPal
* Displays subscription status
* Displays selected payment method
* Instant feedback on button click
* Clean and centered UI design

---

Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

How It Works

* User selects:

  * Subscription checkbox
  * One payment method (radio buttons)
* On clicking **Submit**:

  * JavaScript checks if the user is subscribed
  * Displays subscription message
  * Checks which payment method is selected
  * Displays corresponding payment message
* Messages are updated dynamically using `textContent`
